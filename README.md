# Агуулахын систем

1. Product/бараа/ - барааны үндсэн мэдээллийг хадгална. /барааны код, нэр/

2. Receipt/падаан/ - аль бараа хэдэн ширхэг орсон, гарсныг хадгална. Энэ нь Income/орлого/ болон Issue/зарлага/ class-уудтай хамт ажиллана.

3. Income/орлого/ - барааг агуулахад орлогоор хүлээн авсан мэдээллийг хадгална. Орлогын дугаар, огноо, нийлүүлэгч, мөн орсон бараануудын жагсаалт зэргийг агуулна. Мөн орлогын падаан хэвлэх үйлдэлтэй.

4. Expense/зарлага/ - барааг агуулахаас зарлагаар гаргасан мэдээллийг хадгална. Зарлагын дугаар, огноо, хүлээн авагч, зарагдсан бараануудын жагсаалт зэрэг мэдээллийг агуулна. Мөн зарлагын падаан хэвлэх үйлдэлтэй.

5. InventoryReport/нөөцийн тайлан/ - агуулахын барааны үлдэгдлийг хадгалан тайланг гаргана. Энэ class нь үлдэгдлийг нэмэх, хасах, шалгах үйлдлүүдийг тодорхойлдог. Тайлан нь бүх бараагаар эсвэл сонгосон бараануудын хүрээнд гарч болно. Мөн тайлан хэвлэх үйлдэлтэй./Агуулахын үлдэгдэл шалгах хэсэг/

6. Warehouse/агуулах/ - агуулахын мэдээллийг хадгална. Агуулах бүр өөрийн нярав, үлдэгдлийн бүртгэл, орлогын падаанууд, зарлагын падаануудтай байна. Энэ class нь тухайн барааны үлдэгдлийг хайх, орлого болон зарлагын мэдээллийг бүртгэхэд ашиглагдана.

7. Storekeeper/нярав/ - барааг орлогоор хүлээн авах, зарлага гаргах, үлдэгдлийн тайлан харах, няравын тайлан гаргах, тооллого хийх зэрэг үйлдлүүдийг гүйцэтгэнэ. 

9. StorekeeperReport/няравын тайлан/ - тодорхой хугацааны тайланг гаргана. Энэхүү тайланд эхний үлдэгдэл, нийт орлого, нийт зарлага, эцсийн үлдэгдэл гэсэн мэдээллийг бараа тус бүрээр харуулна.

10. StorekeeperReportRow/няравын тайлангийн мөр/ - Нэг бараанд хамаарах эхний үлдэгдэл, нийт орлого, нийт зарлага, эцсийн үлдэгдлийг хадгална.

11. StockCount - барааны тооллогыг илэрхийлнэ. Тооллогын огноо, системийн тоо хэмжээ, бодит тоо хэмжээ, мөн эдгээрийн зөрүүг хадгална. Зөрүү гарсан тохиолдолд тухайн барааны үлдэгдлийг шинэчлэхэд ашиглагдана.

***
Системд бараа эхлээд Product хэлбэрээр бүртгэгдэнэ.

Бараа агуулахад орж ирэх үед Income үүсэж, түүнд харгалзах Receipt (мөрүүд)-үүд бүртгэгдэнэ.
Үүний үр дүнд тухайн барааны үлдэгдэл InventoryReport дээр нэмэгдэнэ.

Бараа агуулахаас гарах үед Expense үүсэж, тухайн барааны үлдэгдэл InventoryReport дээр хасагдана.

Агуулахын нярав нь хүссэн үедээ InventoryReport ашиглан тухайн үеийн бүх барааны үлдэгдлийг шалгаж болно.
Мөн тодорхой хугацаанд StorekeeperReport ашиглан:
- эхний үлдэгдэл
- нийт орлого
- нийт зарлага
- эцсийн үлдэгдэл
зэргийг бараа тус бүрээр харж болно.

Барааны бодит тоо хэмжээг шалгах үед StockCount ашиглан тооллого хийж,
системийн үлдэгдэлтэй харьцуулан зөрүүг тодорхойлно.
Зөрүү гарсан тохиолдолд тухайн барааны үлдэгдлийг шинэчилнэ.
